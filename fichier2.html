Techniques d'Équité Algorithmique

Méthodes pour réduire les biais dans les modèles d'IA

Approches pour l'Équité

Découvrez les différentes techniques utilisées pour améliorer l'équité des modèles d'apprentissage automatique, avec des exemples concrets pour chaque méthode.

1. Repondération (Reweighing)

Ajuste les poids des échantillons pour équilibrer la représentation des groupes.

Exemple concret

Dans un dataset de recrutement avec 80% d'hommes et 20% de femmes, on attribue un poids de 0.625 aux hommes (1/80%) et 2.5 aux femmes (1/20%) pour équilibrer l'influence pendant l'entraînement.

Avant
Hommes: 80%
Femmes: 20%
→
Après
Poids H: 0.625
Poids F: 2.5
Avantages
  • Simple à implémenter
  • Préserve les données originales
Cas d'usage
  • Déséquilibres évidents
  • Première approche

2. Disparate Impact Remover

Transforme les features pour réduire leur corrélation avec l'attribut sensible.

Exemple concret

Pour un modèle de crédit, si le revenu est corrélé au genre, on modifie les valeurs de revenu pour que leur distribution soit similaire entre hommes et femmes, tout en préservant l'ordre global.

Avant
Revenu moyen:
H: 50k€
F: 35k€
→
Après
Revenu ajusté:
H: 45k€
F: 40k€
Avantages
  • Corrige les biais systémiques
  • Préserve l'utilité des données
Cas d'usage
  • Corrélations indésirables
  • Données sensibles

3. Ajustement de seuil

Applique des seuils différents par groupe pour équilibrer les résultats.

Exemple concret

Pour un modèle de prêt bancaire qui surestime le risque des minorités, on baisse le seuil de décision pour elles (ex: 0.4 au lieu de 0.5) et on augmente légèrement pour le groupe majoritaire (0.55).

Seuils
Majorité: 0.55
Minorité: 0.4
→
Résultat
Taux acceptation:
Équilibré
Avantages
  • Aucun réentraînement
  • Contrôle précis
Cas d'usage
  • Modèles déjà déployés
  • Corrections rapides

4. Débiaisage adversarial

Utilise un modèle antagoniste pour supprimer les biais pendant l'entraînement.

Exemple concret

Pour un système de CV scanning, on entraîne simultanément : 1) Un modèle principal qui prédit la compétence 2) Un adversaire qui essaie de deviner le genre à partir des prédictions Le modèle principal apprend à faire des prédictions indiscernables par l'adversaire.

  • Détecte les biais subtils
  • Approche sophistiquée
Cas d'usage
  • Biais complexes
  • Ressources disponibles

Parité Démographique

Les prédictions positives sont également réparties entre les groupes.

Calcul

P(Ŷ=1|A=a) = P(Ŷ=1|A=b)

Ex: Taux d'acceptation de 30% pour les hommes et 30% pour les femmes.

Quand l'utiliser

Quand l'égalité des résultats est cruciale

Limites

Peut masquer des différences de performance réelle

Égalité des Chances

Les taux de vrais positifs sont égaux entre les groupes.

Calcul

P(Ŷ=1|A=a,Y=1) = P(Ŷ=1|A=b,Y=1)

Ex: 80% des candidats qualifiés sont repérés, quel que soit leur groupe.

Quand l'utiliser

Quand la détection des "bons" cas est importante

Limites

Nécessite des labels vérité terrain fiables

Égalité des Taux d'Erreurs

Les taux de vrais positifs et faux positifs sont égaux entre les groupes.

Calcul

P(Ŷ=1|A=a,Y=y) = P(Ŷ=1|A=b,Y=y) pour y ∈ {0,1}

Ex: Mêmes taux d'erreurs pour tous les groupes démographiques.

Quand l'utiliser

Quand les deux types d'erreurs sont importants

Limites

Difficile à satisfaire complètement

Équilibre des Taux de Précision

La précision est égale entre les groupes pour les prédictions positives.

Calcul

P(Y=1|A=a,Ŷ=1) = P(Y=1|A=b,Ŷ=1)

Ex: Parmi les candidats acceptés, 70% sont vraiment qualifiés dans chaque groupe.

Quand l'utiliser

Quand la qualité des prédictions positives compte

Limites

Peut créer d'autres disparités

Comment choisir sa technique ?

Situation Technique recommandée Raison
Déséquilibre évident dans les données Repondération Corrige simplement les déséquilibres de représentation
Corrélations indésirables avec l'attribut sensible Disparate Impact Remover Supprime directement ces corrélations
Modèle déjà entraîné avec biais Ajustement de seuil Permet des corrections sans réentraînement
Biais complexes et subtils Débiaisage adversarial Détecte et corrige les biais non évidents
